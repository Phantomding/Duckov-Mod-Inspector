# 🦆 Duckov Security Scanner (鸭科夫安全卫士)

专为《逃离鸭科夫》玩家设计的恶意 Mod 检测工具。
保护你的存档，拦截闪退炸弹，拒绝伪装病毒。

## ✨ 功能特点
- **🛡️ 伪装病毒检测**：识别伪装成 Mod 的恶意 C++ 原生程序（针对 Scav 1.5 混淆）。
- **💣 逻辑炸弹拦截**：扫描针对 SteamID 的强制闪退代码（针对 Scav 1.8）。
- **🕵️ 违禁品搜身**：检测不应存在的底层 API 调用（如 KERNEL32, VirtualProtect）。
- **⚡ 极速指纹库**：内置 MD5 白名单与黑名单，秒杀已知文件。

## 🚀 如何使用
1. 在右侧 "Releases" 下载最新的 `.exe` 文件。
2. 启动软件（会伴随一个黑色日志窗口，属于正常现象）。
3. 将你的 Mod 文件夹直接拖入蓝色区域。
4. **绿色** = 安全，**红色** = 立即删除！
